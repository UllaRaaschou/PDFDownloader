# PDFDownloader - Betjeningsvejledning

## Sådan bruger du programmet

### Trin 1: Start programmet
Dobbeltklik på `PDFDownloader.exe` eller kør det fra kommandolinjen.

### Trin 2: Angiv filsti til Excel-fil
```
Indtast filsti: 
```
Indtast den fulde sti til din Excel-fil, f.eks.:
```
C:\Mine Dokumenter\downloads.xlsx
```
**Tips:** Du kan trække Excel-filen direkte ind i kommandovinduet - programmet fjerner automatisk anførselstegn.

### Trin 3: Vælg worksheet
```
Hvilket nummer har det aktuelle worksheet?
```
Indtast nummeret på det worksheet du vil bruge:
- `1` = første worksheet
- `2` = andet worksheet
- osv.

### Trin 4: Vælg download-strategi
```
Skal jeg kun downloade filer, hvis de ikke allerede er downloadet tidligere? (j/n):
```
- **j** (ja): Springer over filer der tidligere er blevet downloadet
- **n** (nej): Downloader alle filer, også selv om de er downloadet før

### Trin 5: Vent på download
```
Download starter, tag en kop kaffe
```
Programmet downloader nu alle filer. Dette kan tage tid afhængigt af:
- Antal filer
- Filstørrelse
- Internethastighed

### Trin 6: Færdig!
```
Starting WriteToExcel...
WriteToExcel completed!
Done!!!
```
Programmet er nu færdigt. Tryk Enter for at lukke.
